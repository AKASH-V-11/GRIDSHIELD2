# ⚡ GRIDSHIELD ⚡

**National Intelligent Power Grid Monitoring & Protection Platform**

GRIDSHIELD is an intelligent power grid monitoring and decision-support platform developed using **Python** and **Streamlit**. It provides a modern dashboard to monitor grid conditions, simulate faults, visualize analytics, and support operators with AI-assisted decision making.

---

## 🚀 Features

- 🔐 Secure Login Interface
- 📊 Interactive Dashboard
- ⚡ Fault Simulation
- 🧠 AI Decision Support
- 🚨 Incident Management
- 🚑 Emergency Dispatch
- 📈 Grid Analytics
- 🗺️ India Power Grid Visualization

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- Folium
- NumPy
